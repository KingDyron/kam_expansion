Campaign Builder tool

Description:
  Campaign Builder allows you to prepare addon campaigns for KaM Remake.

How to use:
  Load picture to be a background image
  Set campaign unique short title (3 letters)
  Set map count
  Select maps in list and position them on image with dragging and/or cursor keys
  Set selected map nodes count and position them on image with dragging and/or cursor keys
  Set selected map default briefing text position
  Check that everything is correct and save campaign
